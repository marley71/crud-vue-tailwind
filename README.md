# crud-vue-tailwind
Pacchetto crud basato su tailwind

## Project setup
```
npm install
```

### create and edit vue.config.js
```
copy vue.config-example.js vue.config.js
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
